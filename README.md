# Nadezhda Gong Meditation

Готовый React/Vite сайт для публикации на Vercel.

## Как запустить локально

```bash
npm install
npm run dev
```

## Как собрать

```bash
npm run build
```

## Публикация

1. Загрузить папку в GitHub.
2. Подключить репозиторий в Vercel.
3. В настройках Vercel добавить домен.
4. У регистратора домена прописать DNS, которые покажет Vercel.
